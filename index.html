• Propiedad Color : (color)

    => Especifica que color tomara la etiqueta

  • Propiedad Font Size : (font-size)

    => Especifica elTamaño de Fuente

  • Propiedad Font Weight : (font-weight)

    => Especifica el Grosor de la fuente

  • Propiedad Text Transform : (text-transform)

    => Especifica el formato que tendra el texto puede ser : Capitalize , UpperCase , LowerCase

  • Propiedad Text Align : (text-align)

    => Especifica el aliniamiento del texto : start center justify left right

  • Propiedad Text Decoration : (text-decoration)

    => Especifica si la etiqueta tendra un borde con border || sin borde
